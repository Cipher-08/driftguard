# DriftGuard

DriftGuard is a unified dashboard for cloud drift detection and compliance monitoring. It supports connecting cloud accounts (AWS, GCP, Azure) and visualizes resource drifts, compliance status, and remediation options.

## Features
- Connect your GCP account securely
- Dashboard with drift and compliance summary
- View resources and detected drifts
- Modern, user-friendly UI (React + Tailwind)
- Extensible backend (Go, Gin, Postgres)

## Getting Started
1. Clone the repo (after first push)
2. Start the backend (Go)
3. Start the frontend (Vite/React)
4. Add your cloud credentials via the UI
